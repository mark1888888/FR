📊 批量匯入支出

使用方式
1. 下載範本 Excel（已含表頭與一筆範例）
2. 填入資料（每列一筆，可一次數十筆）
3. 上傳填好的檔案，預覽並確認
4. 按「全部建檔」一鍵建立所有支出紀錄
欄位說明：
• 日期（必填）：YYYY-MM-DD，或 Excel 日期格式
• 類別（必填）：若未在現有類別則自動新增
• 金額（必填）：數字
• 幣別：TWD / USD / JPY / EUR / CNY / HKD / KRW，預設 TWD
• 帳戶：輸入帳戶名稱（需先在資產頁建立），找不到則使用第一個可用帳戶
• 支付方式：信用卡 / 現金 / 銀行轉帳 / 電子支付，預設「現金」
• 說明、支付對象、使用對象、專案（選填）
