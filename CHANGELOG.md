# FlowRich 更新記錄

## v1.5.0 — 2026-04-16

### 新功能
- **應收款/應付款明細記錄系統**：
  - 在資產管理的「未收款」「應付款」分頁下新增明細記錄功能
  - 可輸入建立日期、應付日期（到期日）、對象/機構、金額、幣別、備註
  - 自動計算逾期天數，以紅色標籤顯示「逾期 X 天」
  - 狀態標籤：未到期（藍）、今日到期（橘）、逾期（紅）、已收款（綠）
  - 支援標記已收款、編輯、刪除操作
  - 資料模型新增 `receivables` 陣列，向下相容
- **支出類別子類別分析彈窗**：
  - 收支分析頁的支出/收入類別圓餅圖圖例可點擊
  - 點擊後彈窗顯示該大分類下的子類別圓餅圖分佈與明細列表
- **新增幣種**：韓元（KRW）、港幣（HKD），匯率自動從 API 取得

### 改善
- **篩選年月下拉選單排序優化**：最近的月份排在最上面（原為最舊在上）
- 年度下拉維持最新年份在上

## v1.4.0 — 2026-04-16

### 新功能
- **表格排序功能**：收入/支出列表的欄位標題可點擊排序（日期、類別、說明、支付對象、使用對象、金額、幣別、支付方式），再次點擊切換升序/降序，標題旁顯示 ▲▼ 排序指示
- **子類別系統**：收支類別支援建立子類別（如「教育 > 學費」「教育 > 午餐費」等）
  - 設定頁面每個大分類旁新增「＋」按鈕，可展開輸入框新增/刪除子類別
  - 新增收支時的類別下拉選單顯示大分類與縮排子類別
  - 資料儲存格式 `"大分類 > 子分類"`，表格中以小字標籤顯示
  - 分析圖表（圓餅圖）以大分類統計，方便總覽
  - 資料模型新增 `subCategories` 欄位，向下相容既有資料
- **收支分析新增支付對象與使用對象分佈圓餅圖**：合併收入+支出統計，空白值歸類為「其他」
  - 新增 `groupByField()` 通用分組函式
  - `drawPie()` 支援 `noIcon` 參數，對象圖表不顯示類別圖示

### 改善
- **股票追蹤 API 改為 TWSE 臺灣證交所**：
  - 移除 Yahoo Finance API 依賴，改為完全使用 `mis.twse.com.tw` 作為唯一資料來源
  - 一次性同時查詢上市（tse）與上櫃（otc），自動去重
  - 個別 fallback 查詢確保不漏掉任何股票
  - 提取 `parseTWSE()` 和 `renderCards()` 輔助函式，消除重複程式碼
- **OCR 帳單掃描大幅強化**：
  - 影像前處理：灰階化 + 對比增強 + Otsu 自適應二值化（取代固定閾值），自動適應不同亮度帳單；大圖自動縮放
  - 文字清理 `_cleanOcrText()`：移除 CJK 字元間空格（臺 北 市 → 臺北市）、合併被空白切開的數字序列、修正日期斜線空格、全形→半形轉換、常見 OCR 數字誤判修正（O→0, l/I→1）、CJK 與英數間不必要空格移除
  - 金額辨識強化：支援「元」「圓」後綴、更積極的字母→數字修正、過濾短數字避免與日期混淆
  - Tesseract 參數優化：`preserve_interword_spaces: '0'`、`SINGLE_BLOCK` 頁面分段模式
- **民國年日期支援**：`_parseDateStr()` 新增 2-3 位數年份辨識（如 `115/04/13` → `2026-04-13`），`_extractDates()` 新增對應正則

## v1.3.0 — 2026-04-16

### 新功能
- **帳戶轉帳功能**：新增「轉帳」頁面，支援帳戶間轉帳（如繳信用卡帳單），避免重複記帳
  - 轉出/轉入帳戶選擇、金額、幣別、備註
  - 自動更新雙方帳戶餘額
  - 轉帳記錄列表含篩選（月份/年度/全部）
- **支出表單「轉入帳號」欄位**：當支付方式選擇「銀行轉帳」時，自動顯示轉入帳號輸入欄位，方便記錄匯款對象帳號
- 側邊欄新增「轉帳」導航項目

### 改善
- **資產分析圖表優化**：
  - 所有圓餅圖圖例新增金額顯示（不再只顯示百分比）
  - 各帳戶餘額排行排除信用卡帳戶（信用卡屬於負債，不應列在正資產排行中）
  - 新增「信用卡負債排行」和「應付款排行」橫條圖，與帳戶餘額排行一致風格
- **專案資料夾結構重整**：
  - `app.js` → `js/app.js`
  - `style.css` → `css/style.css`
  - 圖片檔案（favicon、logo 等）→ `images/`
  - SQL 設定檔 → `sql/`
  - `index.html`、`admin.html` 保留於根目錄
- 所有檔案引用路徑已同步更新

## v1.1.0 — 2026-04-15

### 新功能
- **Supabase 雲端同步**：支援多裝置資料同步
- **使用者登入/註冊**：Email + 密碼驗證
- **管理後台** (`admin.html`)：使用者管理、資料檢視

### 基礎功能
- 收入/支出記帳，支援多帳戶、多幣別
- 資產管理（銀行、現金、信用卡、未收款、應付款、投資）
- 分析報表（圓餅圖、月趨勢）
- 投資觀察清單（台股即時報價）
- 匯率自動轉換（TWD/USD/JPY/EUR/CNY）
