# FlowRich 更新記錄

## v1.3.0 — 2026-04-16

### 新功能
- **帳戶轉帳功能**：新增「轉帳」頁面，支援帳戶間轉帳（如繳信用卡帳單），避免重複記帳
  - 轉出/轉入帳戶選擇、金額、幣別、備註
  - 自動更新雙方帳戶餘額
  - 轉帳記錄列表含篩選（月份/年度/全部）
- **支出表單「轉入帳號」欄位**：當支付方式選擇「銀行轉帳」時，自動顯示轉入帳號輸入欄位，方便記錄匯款對象帳號
- 側邊欄新增「轉帳」導航項目

### 改善
- **專案資料夾結構重整**：
  - `app.js` → `js/app.js`
  - `style.css` → `css/style.css`
  - 圖片檔案（favicon、logo 等）→ `images/`
  - SQL 設定檔 → `sql/`
  - `index.html`、`admin.html` 保留於根目錄
- 所有檔案引用路徑已同步更新

---

## v1.2.0 — 2026-04-16

### 新功能
- **OCR 帳單掃描**：支援上傳信用卡帳單圖片，使用 Tesseract.js v5 辨識消費明細（消費日、說明→類別、台幣金額），自動填入支出表單
- **自訂類別圖示**：支出/收入類別可自訂 emoji 圖示

### 改善
- 彈窗改為 HTML Modal（取代 native alert/confirm）
- Toast 通知系統取代 native alert()
- 類別與說明欄位寬度放寬

---

## v1.1.0 — 2026-04-15

### 新功能
- **Supabase 雲端同步**：支援多裝置資料同步
- **使用者登入/註冊**：Email + 密碼驗證
- **管理後台** (`admin.html`)：使用者管理、資料檢視

### 基礎功能
- 收入/支出記帳，支援多帳戶、多幣別
- 資產管理（銀行、現金、信用卡、未收款、應付款、投資）
- 分析報表（圓餅圖、月趨勢）
- 投資觀察清單（台股即時報價）
- 匯率自動轉換（TWD/USD/JPY/EUR/CNY）
